📱 Guide Complet : Setup Téléphone Virtuel dans VS Code

1️⃣

Installer Live Server

L'extension Live Server vous permet de lancer un serveur local avec reload automatique.

  1. Dans VS Code, appuyez sur Ctrl+Shift+X
  2. Recherchez Live Server
  3. Installez l'extension par Ritwick Dey
  4. Redémarrez VS Code (optionnel)
✅ Quand c'est fait, vous verrez "Go Live" en bas à droite de VS Code
2️⃣

Lancer le serveur local

Il y a deux façons de démarrer Live Server :

Méthode 1 (la plus simple) :

  1. Ouvrez index.html dans VS Code
  2. Clic droit sur le fichier
  3. Sélectionnez "Open with Live Server"

Méthode 2 (rapide) :

  1. En bas à droite de VS Code, cliquez sur "Go Live"
✅ Chrome s'ouvre automatiquement à http://localhost:5500
3️⃣

Diviser l'écran VS Code

Pour avoir l'éditeur et le téléphone côte à côte :

  1. Dans VS Code, menu View
  2. Sélectionnez Editor Layout
  3. Choisissez Split Right (Colonne 2)
┌─────────────────────────────┬──────────────────────────┐ │ COLONNE 1 (CODE) │ COLONNE 2 (NAVIGATEUR) │ │ │ │ │ VS Code - index.html │ Chrome avec mode mobile │ │ VS Code - style.css │ Aperçu du téléphone │ │ VS Code - script.js │ │ │ │ │ └─────────────────────────────┴──────────────────────────┘
ℹ️ Si vous n'avez qu'un seul écran, c'est OK ! Réduisez juste les deux fenêtres côte à côte.
4️⃣

Activer le mode mobile dans Chrome

Transformez Chrome en téléphone virtuel :

  1. Dans la fenêtre Chrome, appuyez sur F12
  2. Appuyez sur Ctrl+Shift+M (ou cliquez sur l'icône téléphone)
  3. En haut à gauche, sélectionnez un appareil :
Suggestions : • iPhone 12 Pro (390 × 844) • iPhone SE (375 × 667) • Pixel 6 Pro (412 × 915) • Galaxy S21 (360 × 800)
✅ Maintenant vous avez un vrai téléphone virtuel à droite !
5️⃣

Testez le workflow

Vérifions que tout fonctionne :

  1. À gauche dans VS Code, ouvrez index.html
  2. Modifiez quelque chose de visible (ex: changez le titre ou une couleur)
  3. Sauvegardez avec Ctrl+S
  4. Regardez le téléphone à droite → ça se met à jour tout seul !
🎉 Bravo ! Vous avez le setup professionnel !

Checklist de vérification :

  • VS Code ouvert avec le projet
  • Live Server en cours d'exécution
  • Chrome ouvert avec mode mobile activé
  • Écran divisé en 2 colonnes
  • Test : modification = mise à jour auto
⚡

Raccourcis clavier essentiels

Ctrl+S Sauvegarde → Auto-reload
F12 Ouvre DevTools Chrome
Ctrl+Shift+M Toggle mode responsive
Ctrl+Shift+X Ouvre les extensions VS Code
Ctrl+` Ouvre le terminal VS Code
Ctrl+B Toggle la barre latérale
🔧

Dépannage

❌ Chrome ne s'ouvre pas ?

  • Cliquez manuellement sur le lien http://localhost:5500
  • Ou ouvrez Chrome et allez à http://localhost:5500

❌ Le mode responsive ne marche pas ?

  • Appuyez sur F12 pour ouvrir DevTools
  • Puis Ctrl+Shift+M pour activer le responsive mode

❌ La page ne se met pas à jour ?

  • Appuyez sur Ctrl+Shift+R pour forcer le reload
  • Ou cliquez sur le bouton reload dans la barre d'outils

❌ Port 5500 déjà utilisé ?

  • Changez le port dans les paramètres de Live Server
  • Ou tuez le processus : lsof -i :5500
