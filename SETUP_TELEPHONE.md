# 📱 SETUP ULTRA-SIMPLE : Téléphone Virtuel

## ⚡ 3 Étapes (1 minute)

### Étape 1️⃣ : Lancer le serveur
Dans VS Code, cherchez `index.html` dans l'Explorateur de fichiers :
- **Clic droit** → **"Open with Live Server"**

Chrome s'ouvre automatiquement à `http://localhost:5500`

### Étape 2️⃣ : Ouvrir l'aperçu mobile
Dans le navigateur qui vient d'ouvrir, allez à :
```
http://localhost:5500/mobile-preview.html
```

Vous voyez maintenant un **téléphone virtuel** qui affiche votre site ! 📱

### Étape 3️⃣ : Disposition finale
1. **Gardez VS Code ouvert** (votre code sur la gauche)
2. **Réduisez la fenêtre Chrome** (mettez le téléphone à droite)
3. **Redimensionnez** pour avoir côte à côte

```
┌─ VS CODE ─────────┬─ CHROME (Téléphone) ─────┐
│ index.html        │ 📱 Aperçu mobile          │
│ style.css         │ (mise à jour auto)        │
│ script.js         │                           │
└───────────────────┴──────────────────────────┘
```

---

## 🎮 Utilisation

### Avant chaque modification
**VS Code** → Modifiez votre HTML/CSS/JS

### Après modification
**Ctrl+S** (sauvegarder) → Le téléphone se met à jour automatiquement ✨

### Si ça ne se met pas à jour
- Cliquez sur le bouton **🔄 Recharger** dans le téléphone
- Ou changez d'appareil (iPhone → Pixel) et revenez

---

## 📱 Appareils disponibles

L'aperçu mobile supporte :
- **iPhone 14** - 390×844px
- **Pixel 7** - 412×915px  
- **iPad** - 600×800px

Cliquez sur les boutons pour changer !
