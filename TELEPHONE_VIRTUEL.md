# 📱 SETUP : Téléphone Virtuel dans VS Code

## ✅ L'extension est maintenant installée !

### 🚀 Pour démarrer immédiatement :

1. **Redémarrez VS Code** (ou rechargez la fenêtre : Ctrl+Shift+P → Reload Window)

2. **Ouvrez l'aperçu mobile** avec **Ctrl+Shift+P** :
   - Tapez : `📱 Ouvrir Aperçu Mobile`
   - Appuyez sur Entrée
   - Le téléphone virtuel s'affiche dans le panneau de droite

3. **Commencez à coder** :
   - Modifiez votre `index.html`, `style.css`, ou `script.js`
   - Appuyez sur **Ctrl+S** pour sauvegarder
   - Le téléphone se met à jour automatiquement ! 📱

L'extension supporte 3 types d'appareils :

## 🛠️ Dépannage

### ❌ Le téléphone reste blanc ?
- Vérifiez que votre fichier `index.html` existe
- Cliquez sur le bouton "🔄 Recharger"
- Vérifiez la console VS Code (Ctrl+`) pour les erreurs

### ❌ Le reload ne marche pas ?
- Cliquez sur le bouton "🔄 Recharger" manuellement
- Ou changez l'URL dans la barre d'adresse

### ❌ Impossible d'ouvrir l'aperçu ?
- Redémarrez VS Code complètement
- Allez à Ctrl+Shift+P → Reload Window
- Puis relancez la commande
